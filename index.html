Docs

            
Developer Quickstart

            

                

                    
Install SDK

                    
npm i rotec-ai
# or
pip install rotec-ai

                    
Use in JavaScript

                    
import { Rotec } from 'rotec-ai'
const rt = new Rotec({ apiKey: process.env.ROTEC_KEY })
const out = await rt.chat({ model:'phantom', messages:[{role:'user', content:'Hello'}] })
console.log(out.choices[0].message)

                    
Use in Python

                    
from rotec_ai import Rotec
rt = Rotec(api_key=ROTEC_KEY)
out = rt.chat(model="phantom", messages=[{"role":"user","content":"Hello"}])
print(out["choices"][0]["message"])

                    
See the full API in the Playground below.

                

                

                    
Auth & Endpoints

                    

                        

                            Base URL
                            https://api.rotec.example/v1
                        

                        

                            Header
                            Authorization: Bearer <key>
                        

                        

                            Models
                            phantom, ghost-agent, wraith-vision
                        

                    

                    
Curl

                    
curl -X POST https://api.rotec.example/v1/chat \
 -H "Authorization: Bearer $ROTEC_KEY" \
 -H "Content-Type: application/json" \
 -d '{
  "model": "phantom",
  "messages":[{"role":"user","content":"Hello"}]
}'
